Help & Documentation

🔍 Search & Filter

Use the search bar to find assets by name, path, or owner (client, project, or brand name, e.g., 'le-cafe-litteraire-de-fes', 'webkenrel', 'thebestrecruit'). Filter by type (Brands, Clients, Projects) using the dropdown.

👁️ View Modes

Switch between grid and table views using the toggle buttons in the header.

📄 Asset Preview

Click on any asset to preview it in a modal. Images, documents, and code files are supported.

💾 Download Assets

Download any asset directly using the download button in the preview modal.

🔗 Share Links

Copy direct links to assets using the "Copy Link" button.
